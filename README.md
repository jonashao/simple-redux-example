# Simple Redux Example
Fetching a post using redux in React.
Inspired by blog：[使用Redux管理你的React应用](https://github.com/matthew-sun/blog/issues/18)

## Getting start
``` bash
npm i -g create-react-app

npm install
npm start
```

## File structure
```
 src
   |-- index.js
   |-- configureStore.js
   |-- constants
   |  `-- actionTypes.js  
   |-- actions
   |  `-- index.js
   |-- reducers
   |  |-- index.js
   |  `-- post.js
   |-- components
   |  `-- Post.js
   `-- containers
      |-- App.js
      `-- DevTools.js
   
```
